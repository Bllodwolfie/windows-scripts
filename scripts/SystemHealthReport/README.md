# SystemHealthReport

Generates a self-contained HTML report of your Windows system: CPU, GPU, memory, storage, network, top processes, recent errors, and Windows Update status.

Features a full-viewport intro splash, Catppuccin dark/light themes, sticky sidebar navigation, and live animated progress bars.

## Usage

```powershell
.\SystemHealthReport.ps1
```

Output: `$env:USERPROFILE\Documents\System_Health_Report.html`

## Preview

Open [`sample-report.html`](sample-report.html) for a preview with dummy data.
